Lileikavimas
Fraktalinis-poetinis generatorius, paremtas Esatis 1.0 / Sarmatiška Relational Ontology.
Vieną sėklos eilutę paverčia ritmingu, daugiasluoksniu semantiniu lauku per sukimą, mutaciją ir prasmių virsmą.
Kaip naudoti
Atsidarykite `lileikavimas-lt.html` bet kurioje naršyklėje (nereikia serverio).
Įveskite sėklos eilutę (tinkamiausia 9 žodžiai).
Pasirinkite struktūrą (9×9 rekomenduojama).
Generuokite ir eksperimentuokite.
Failai
`lileikavimas.html` — pagrindinis įrankis (lietuvių kalba)
